# Core

Contains application-wide infrastructure shared across all features.

## Sub-directories

| Directory     | Purpose                                                                 |
|---------------|-------------------------------------------------------------------------|
| `constants/`  | App-wide constant values — API keys, durations, asset paths, strings.   |
| `theme/`      | Material 3 theme data, color schemes, typography, and component themes. |
| `services/`   | Singleton services — Hive init, notification service, health service.   |
| `utils/`      | Pure utility / helper functions — formatters, validators, converters.   |
| `routing/`    | Centralized route definitions and the app router configuration.         |
